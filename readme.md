### Básicos

* 1.1 var = variavél

``` javascript
    function comando() {

```
* 1.3 Sinais:
    * 1.3.1 + = Soma
    * 1.3.2 - = Subtração
    * 1.3.3 * = Multiplicação
    * 1.3.4 / = Divisão
    * 1.3.5 ++ = Adiciona 1
    * 1.3.6 = = Atribuição
    * 1.3.7 == = Igualdade
    * 1.3.8 === = Idêntico

``` javascript
    var a = 5
    a + 5 // 10
    a - 3 // 7
    a * 2 // 14
    a / 7 // 2
    a++ // 3
    a = 5 // 5
```
